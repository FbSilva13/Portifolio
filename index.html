Sobre mim

Desde pequeno fui fissurado em tecnologia, ganhei meu primeiro computador em 2008 comprado na lojas americanas que vinha com um processador Intel Celeron D, 512mb de ram, 40gb de HD e Windows XP start edition. Lembro muito bem desse sistema operacional pois foi a versão mais basica lançada pela Microsoft do Windows XP e não vinha com opção de mudar o tema, não tinha aquele classico wallpaper verde do Windows XP profissional e nem a classica barra de tarefas azul, e por eu já ter mexido no Windows XP PRO, fiquei encucado de saber o porque que meu Windows XP não tinha aquelas coisas. Então por ser curiso futuquei todo o sistema mexendo em dll em arquivos do sistema, tentei copiar arquivos do outro windows para colocar no meu e ter aquele tema, tentei, tentei, até descobrir que meu Windows XP não tinha todos os arquivos necessarios para fazer funcionar os temas. Foi então que busquei saber como mudava o sistema do meu computador
